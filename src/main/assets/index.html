Introduction

    Stacks Flashcards aims to be an easy to use app to help you learn flashcards quickly. To use the help, use the above links to navigate.

Terminology

  • Stack: A collection of Cards. Each stack has an associated icon, title, and description. The title must be unique.
  • Card: A card contains a title (front side), details (a back side), and an optional image attachment
  • Play Session: Any session in which cards are presented to review, either for learning or testing
  • Importing Sets: Importing from a 3rd party website
