Test Settings

    Session settings allow you to customize a test. Presets allow you to quickly select often used combinations of settings.

Presets:
  • Untimed: No time limit, navigate cards freely
  • Timed: Defaults to 1 minute a card
  • Rapid Fire: Quick tests with only 10 seconds per card
Options:
  • Stack Time Limit: The time limit for the entire test
  • Card Time Limit: The time limit for a single card
  • Card Types: Select which question types to include among Simple, Multiple Choice, and Write-In

Learning and Testing

    Learning and testing modes allow you to review the cards in a flash card format. Testing keeps track of right and wrong answers, but otherwise, both are similar. Sessions have a timer running on top to let you keep track, which can be paused by tapping. Simply tap the check mark or "X" to mark correct or wrong. Swipe to move on. If the session is to end early, tap the red X in the top right corner. By default, all testing sessions will be enabled to view in Stats.
