Settings

Theme

  • Theme: Choose Light, Dark, or Pitch Black to change how Stacks Flashcards looks. Pitch Black will benefit battery life on AMOLED devices

Stacks

  • Title Text Size: Change the size of the title in the Stack window
  • Description Text Size: Change the size of the description in the Stack window
  • Delete On Swipe: If checked, Stacks will be permanently deleted on swipe, instead of archived. USE WITH CAUTION

Cards

  • Title Text Size: Change the size of the title in the Card window
  • Details Text Size: Change the size of the details in the Card window
  • Auto Crop Drawing: When a drawing is saved, extra whitespace will be trimmed if checked.
  • Preview On Tap: Show the Card preview when a Card is tapped in the Card window
  • Delete On Swipe: If checked, Cards will be permanently deleted on swipe, instead of archived. USE WITH CAUTION

Play

Settings will affect Learning and Testing sessions

  • Title Text Size: Change the size of the title in Play sessions
  • Details Text Size: Change the size of the details in Play sessions
  • Shuffle before testing session: Shuffle the stack before being tested if checked, otherwise use the current order
  • Auto Advance: Advance to the next card when correct or incorrect is chosen

Advanced

These settings should normally not be used, unless you know what you are doing!

  • Clean Attachment Directory: This will delete all attachments in the attachments directory that are currently not in use. THIS CANNOT BE UNDONE
  • Clean Empty Members: Delete all Stacks, Cards, and Play Sessions that have no user inputted data
  • Reset Preferences: Reset all preferences to the default values
  • Google Analytics opt-out: Check to stop sending usage metrics to Google Analytics. No personal information is sent, and keeping this unchecked allows rapid, relevant updates! Recommended Off
  • Add a sample Stack: Add the sample stack the application starts out with, if it doesn't already exist

Import and Export

  • Export Stacks File: A file with all current stacks, cards, and play sessions will be exported in the Stacks directory on the sdcard. Do not include an extension
  • Import Stacks File: Import a previously exported file. Do not include the extension, and the file should be in the Stacks directory on the sdcard
  • Backup Frequency: How often to write stacks_bak.stk to the Stacks directory on the SD card, in hours. This will persist even if Stacks is uninstalled
